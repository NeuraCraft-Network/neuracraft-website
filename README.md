# NeuraCraft Website

Official frontend website for **NeuraCraft Minecraft Server Community**.

This repository contains **frontend assets only**, including:
- HTML
- CSS
- Lightweight JavaScript (UI & client-side logic)

Backend services, server-side logic, and internal systems are **NOT included** in this repository.

---

##  About NeuraCraft

NeuraCraft is a Minecraft server community focused on building a creative, friendly, and collaborative environment for players.

This website serves as:
- Community landing page
- Public information portal
- Frontend interface for community features

---

##  Repository Scope

 Included:
